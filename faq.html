What is the Document Object Model (Dom)?

The Document Object Model (DOM) is the data representation of the objects that comprise the structure and content of a document on the web.

How do you select an element from the DOM?

To select an element from the DOM in JavaScript, you can use various methods provided by the document object. Here are some commonly used methods: 1. getElementById() Selects an element by its ID. 2. getElementsByClassName() Selects elements by their class name (returns an HTMLCollection). 3. getElementsByTagName() Selects elements by their tag name (e.g., div, p, etc.). 4. querySelector() Selects the first element that matches a CSS selector. 5.querySelectorAll() Selects all elements that match a CSS selector (returns a NodeList):

What is the event delegation in the context of DOM, and why is it useful

Event delegation is a technique in the context of the DOM where you leverage event bubbling to handle events more efficiently. Instead of adding event listeners to multiple child elements, you add a single event listener to a common parent element. The parent element catches events triggered by the child elements due to event propagation (bubbling up the DOM tree). How Event Delegation Works: When an event is triggered on a child element, it first triggers on the child and then bubbles up to its parent, continuing up to the root (document). By attaching the event listener to the parent element, you can capture the event and handle it for any of the child elements.
Why Event Delegation Is Useful: Efficiency: It reduces the number of event listeners needed, improving performance, especially when dealing with a large number of child elements. Dynamic Elements: It's useful for handling events on dynamically added elements, as you don't need to re-attach event listeners to new elements. Cleaner Code: You manage all similar events in one place rather than attaching separate listeners to each element.

How do you manipulate an event's attributes and styles using DOM?

We can manipulate an event's attributes and styles by accessing the event object, which provides information about the event and its target element. You can change styles like background color or font size directly on the element that triggered the event. Additionally, you can modify attributes like text or disabling a button, and control the event's behavior using methods like preventDefault() to stop default actions.
